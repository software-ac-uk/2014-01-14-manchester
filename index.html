MATLAB

The attendees will need access to MATLAB R2013b and Statistics Toolbox. Trial licenses for all products will be available for the duration of the training.

As long as the attendees have a valid academic email address they should be able to access the online tutorials. There is a lot of material available, so even if you have some previous experience of MATLAB, it should be possible to find something useful there prior to the course.
